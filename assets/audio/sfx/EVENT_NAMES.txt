Per-gun sounds:
- Each gun now has a unique `sfx` key in scripts/data/weapon_db.gd, e.g. `gun_ak_47`.
- Preferred file location convention:
	assets/audio/sfx/guns/<event_without_gun_prefix>.ogg
	Example: event `gun_ak_47` -> `assets/audio/sfx/guns/ak_47.ogg`

You can rename or remap events in scripts/autoload/audio_manager.gd:
- DEFAULT_EVENT_PATHS dictionary
- set_event_path(event_name, path)
- register_bank({event_name: path, ...})
